# 📊 New Orleans Sewage & Water Board Dashboard 
### Infrastructure Project Analysis (SSERP)

---

## 📌 Overview

This project presents an interactive Tableau dashboard analyzing infrastructure projects under the **Sanitary Sewer Evaluation and Rehabilitation Program (SSERP)** in New Orleans.

The dashboard provides a comprehensive view of project progress, service distribution, and geographic impact, enabling users to understand how sewer system improvements are being implemented across the city.

---

## 🎯 Objectives

- Analyze the lifecycle of infrastructure projects (Planned → Active → Completed)
- Identify dominant service types (e.g., sewer line repair, drainage systems)
- Visualize geographic areas with the highest infrastructure impact
- Communicate key insights through a clean, executive-style dashboard

---

## 📊 Dashboard Features

### 🔢 Key Performance Indicators (KPIs)
- Total Projects: **429**
- Planned Projects: **182**
- Active Projects: **3**
- Completed Projects: **336**

---

### 📈 Visualizations

- **Projects by Status** 
  Breakdown of project stages (Completed, Planned, Construction)

- **Projects by Service Type** 
  Distribution of infrastructure work (sewer lines, drainage, water lines)

- **Project Status Distribution (Pie Chart)** 
  Proportional view of project completion stages

- **Infrastructure Impact Map** 
  Geographic visualization highlighting high-density repair zones across New Orleans

---

### 🧠 Key Insights

- SSERP is driving **429 total infrastructure projects** across New Orleans 
- **336 projects completed**, showing strong rehabilitation progress 
- **182 planned projects** indicate sustained infrastructure investment 
- Only **3 active projects**, suggesting a transition between project phases 
- Work is concentrated in **sewer and drainage line repairs** 
- High-impact zones are primarily located in **central New Orleans**
